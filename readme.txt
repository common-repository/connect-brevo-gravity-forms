When someone submits a form on your site, it sends form submissions from Gravity Forms to the relationship marketing platform Brevo (ex Sendinblue).

== Description ==
This plugin connects to brevo API to send gravity forms submitted to into brevo.

1. This plug in allow you to add or subscribe users to Brevo's contact list with gravity forms.

2. Brevo offical website: https://www.brevo.com/

3. To achieve this, the pluin using Brevo list's API (you can read more here: https://help.brevo.com/hc/en-us/articles/209467485-Create-and-manage-your-API-keys)

== Installation ==

1. Best is to install directly from WordPress.

2. Activate the plugin through the 'Plugins' menu in WordPress
